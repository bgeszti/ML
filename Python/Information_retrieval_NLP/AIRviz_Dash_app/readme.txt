The application provides a dashboard to explore the performance of IR models. 

***Web application***
If you don't want to run the application locally, use the following link to an online web-enabled visualization:
Open https://twentytwo-air.herokuapp.com/ in your browser!

***Running locally - requirements***
To run the application Phyton 3 should be installed. 

To install all of the required Phyton packages, simply run:
pip install -r requirements.txt

***How to use the app***:
Run this app locally by:
python app.py
Open http://127.0.0.1:8050/ in your browser.

Use the dropdown boxes to select models, measurements and queries. Use the graph's menu bar to select, zoom or save the plots.
